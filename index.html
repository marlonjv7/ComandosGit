Paso a paso para trabajar con ramas

  • Crear o clonar el repositorio en la rama master
  • git pull: antes de crear crear el branch o la rama para verificar si alguien mas realizó un cambio
  • git branch: para verificar las ramas que están localmente
  • git checkout -b nombreRama: para crear y dirigirnos a la rama
  • si la rama está creada ejecutar git checkout nombreRama
  • git add .: Despúes que se hayan realizado los cambios locales
  • git status
  • git commit -m "Agregando el mensaje de la actualización"
  • git push --set upstream origin nombreRama
  • Después del push en la terminal aparece un link, donde nos tenemos que dirigir
  • Nos dirigimos al repositorio (link que emitió el git push)
  • Enviamos el pullrequest
  • verificamos que no hayan conflictos
  • Enviamos el merge todo esto en el mismo repositorio
